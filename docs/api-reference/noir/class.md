# Class

```lua
Noir.Class(name, ...)
```
Create a class that objects can be created from.

Note that classes can inherit from other classes.

### Parameters
- `name`: string
- `...`: NoirClass
### Returns
- `NoirClass`

---

```lua
Noir.IsClass(object)
```
Returns if the provided argument is a class or not.

### Parameters
- `object`: any
### Returns
- `boolean`
